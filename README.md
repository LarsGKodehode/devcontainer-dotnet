## How
1. Fork this repo
2. Docker ? git clone repo-url
2. Open in codespaces

3. Create Blazor + WASM example application
```sh
dotnet new blazorwasm --name <app-name>
```
4. Move working directory to new directory
```sh
cd <app-name>
```
5. Start application
```sh
dotnet run
```
